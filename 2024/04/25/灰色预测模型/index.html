灰色预测模型的基本定义

灰色预测是一种含有不确定因素的因素进行预测的方法，通常是在一定范围内变化的，与时间有关的灰色过程进行预测

灰色预测的常见类型

  • 灰度时间序列预测
  • 畸变预测
  • 系统预测
  • 拓扑预测

一. 灰色关联度分析

优点:
一. 不要求大量的数据
二. 计算量先对较小
三. 出现反常的概率较低

步骤

  1. 选取参考数列
    这个数列的选择就是每一个栏目的最好情况组成的数列

假设此时有m个比较数列
$$
X_i = {X_i(k)|k = 1, 2, …, n} = {X_i(1), … , X_i(n)} (i = 1, 2, … , m)
$$
2. 计算关联系数

这个是比较数列$X_i$对参考数列$X_0$在k时刻的关联系数
其中 $\rho$ $\in$ $[0, +inf]$, $\rho$越大，分辨率越大，反之越小

  1. 计算关联度
    $$
    r_i = \frac{1}{n}\sum_{k = 1}^{n}\zeta_i(k)
    $$
  2. 正相关和负相关的计算
    $$
    \sigma_i = \sum_{k = 1}^{n} {kX_i(k)} - \sum_{k = 1}^{n}{X_i(k)}\sum_{k = 1}^{n}\frac{k}{n}, \
    \sigma_n = \sum_{k = 1}^{n} {k^{2}} - \frac{\sum_{k = 1}^{n}{k^{2}} }{n}
    $$
    如果$$ sign(\frac{\sigma_i}{\sigma_n}) = sign(\frac{\sigma_j}{\sigma_n})$$则$X_i$就是和$X_j$正相关
    如果$$ sign(\frac{\sigma_i}{\sigma_n}) = -sign(\frac{\sigma_j}{\sigma_n})$$则$X_i$就是和$X_j$负相关

案例分析

利用灰色关联分析对6位教师工作状况进行综合分析

分析指标包括：专业素质、外语水平、教学工作量、科研成果、论文、著作与出勤．

数据表格如下

  1. 首先确定参考数列
    $$ {x_0} = {9, 9, 9, 9, 8, 9, 9} $$
  2. 计算 |$x_0(k) - x_j(k)$| 得到下表格
  3. 求最值
    $$
    \min_{i = 1}^{n}\min_{k = 1}^{m}|x_0(k) - x_i(k)| = \min(0, 1, 0, 1, 0, 0) = 0 \
    \max{i = 1}^{n}\max{k = 1}^{m}|x_0(k) - x_i(k)| = \max(7, 6, 5, 6, 6, 5) = 7 \
    $$
  4. 根据公式，取$\rho$ = 0.5得到下列式子
    $$
    \begin{align*}
    &\zeta_1(1) = \frac{0 + 0.5\times{7}}{1 + 0.5\times{7}} = 0.778& \
    &\zeta_1(2) = \frac{0 + 0.5\times{7}}{0 + 0.5\times{7}} = 1.000& \
    &\zeta_1(3) = 0.778& \
    &\zeta_1(4) = 0.636& \
    &\zeta_1(5) = 0.467& \
    &\zeta_1(6) = 0.333& \
    &\zeta_1(7) = 1.000& \
    \end{align*}
    $$
  5. 同理得出其他值
  6. 分别计算每个人指标关联系数的均值
    $$
    \begin{agin}
    & r_{01} = \frac{0.778 + 1.000 + 0.778 + 0.636 + 0.467 + 0.333 + 1.000}{7} = 0.703& \
    & r_2 = 0.614, r_3 = 0.680, r_4 = 0.599, r_5 = 0.683, r_6 = 0.658.
    \end{agin}
    $$
  7. 根据$r_0$排序，得出最终的结果
